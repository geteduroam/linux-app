# geteduroam Linux client

This repository contains the source code for the geteduroam Linux client. Currently WIP.

# Dependencies
- Go >= 1.18
- Make
- GTK >= 4.06 (for the GUI)
- Libadwaita >= 1.1 (for the GUI)

# Building
## CLI
```bash
make build-cli
```

## GUI
```bash
make build-gui
```

# Running
## CLI
```bash
make run-cli
```

## GUI
```bash
make run-gui
```

# Testing
```bash
make test
```
